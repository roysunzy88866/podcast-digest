## 二、四个页面（结构已定稿，视觉全归你）

1. **列表页**（知识入口）：单集卡流 + 搜索 + 标签筛选 + 知识库导航（人/司/概念/图谱）。
   单集卡元素：封面图 / 标题 / 嘉宾+身份 / **一句精华语录（视觉焦点）** / 来源+日期 / 已读淡标记。
   ⚠️ 结构关键：「标签」（筛本页）和「知识库导航」（跳别处）是两类操作，需视觉上分层——用户明确纠正过。
2. **单集详情页**（读这个）：中文精华 + 音频 + **三段式知识关联区**（①顶部类型化关系 ②局部关系图 ③相关单集）。
   精华里 AI 补充的背景 vs 嘉宾原话，**需视觉上明确区分**（用户核心诉求：好懂但不混淆哪是原话）。
3. **实体页**（人物/公司/概念共用一套模板）：头部 + 金句墙 + 出现在哪些集 + 关联实体 + 关系图。
4. **图谱页**：全局关系网（不同类型不同颜色 + 图例）。
